### Installation
1. Clone the repo
   ```sha
   git clone git@github.com:BubbleKID/No-a-bad-code-test.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run dev server
   ```sh
   npm run dev
   ```
